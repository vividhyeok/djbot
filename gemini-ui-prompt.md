# AutoMix DJ Bot — UI 리디자인 프롬프트 (for Gemini)

---

## 프롬프트

아래는 Tauri + 웹 프론트엔드로 만든 데스크탑 앱 AutoMix DJ Bot의 현재 UI야.
전체 UI를 리디자인해줘. 요구사항은 다음과 같아.

---

### 디자인 방향

**레퍼런스 스타일**: 토스(Toss) 앱
- 과한 설명 텍스트 없이, UI 요소 자체가 직관적으로 용도를 설명하는 방식
- 불필요한 안내 문구 전부 제거. 기능이 명확하면 설명하지 않는다
- 여백을 충분히 줘서 답답하지 않게
- 카드, 버튼, 인풋 등 컴포넌트에 그림자보다 경계선 또는 배경 대비로 구분
- 애니메이션은 쓴다면 subtle하게. 화려한 transition 금지

**컬러 팔레트** (아래 색을 기반으로 tone & manner 구성):
```
--color-darkest:  #41431B  /* 가장 어두운 올리브 그린 — 배경 베이스 */
--color-mid:      #AEB784  /* 중간 톤 — 보조 텍스트, 비활성 요소 */
--color-light:    #E3DBBB  /* 밝은 베이지 — 카드 배경, 구분선 */
--color-lightest: #F8F3E1  /* 가장 밝은 크림 — 주요 텍스트, 강조 */
```
전체 배경은 `#41431B` 기반 어두운 계열. 텍스트와 포인트는 밝은 크림/베이지 계열.
보라/파랑/네온 계열 색상은 일절 사용하지 않는다.

**폰트**: 한글은 Pretendard, 영문은 DM Sans 또는 Syne 사용. Inter/Roboto 금지.

### 현재 UI 구조 (변경 전)

왼쪽 패널 (입력 영역):
- YouTube URL 입력 + 숫자 인풋(최대 곡 수) + 다운로드 버튼
- "or" 구분선
- MP3/WAV 드래그앤드롭 영역 + 파일 선택 버튼

오른쪽 패널 (상태 영역):
- 음악 추가 전: 중앙에 아이콘 + "왼쪽에서 음악을 추가하세요" + 설명 텍스트

헤더:
- 헤드폰 이모지 아이콘 + "AutoMix DJ Bot" 타이틀 + 부제목 + "준비됨" 상태 뱃지

---

### 변경 요구사항

**제거할 것**
- "왼쪽에서 음악을 추가하세요" 텍스트
- "YouTube URL 또는 로컬 파일을 추가하면 AI가 자동으로 믹스를 생성합니다." 설명
- "AI-Powered Automatic Mix Generator" 부제목
- "YouTube 플리 가져오기" 섹션 라벨 텍스트

**유지할 것**
- YouTube URL 입력 + 곡 수 설정 + 다운로드 액션
- 파일 드래그앤드롭 + 파일 선택
- 헤더의 앱 이름 + 상태 표시
- 왼쪽/오른쪽 2패널 레이아웃

**개선할 것**
- 드래그앤드롭 영역: 테두리 점선 + 아이콘 + "MP3 / WAV" 레이블만 남기고 나머지 제거
- 오른쪽 빈 상태: 텍스트 없이 시각적 요소(아이콘 또는 일러스트)만으로 표현
- YouTube 입력과 파일 업로드 사이 "or" 구분은 얇은 선 + 소문자 "or" 텍스트로 미니멀하게
- 버튼: 배경색 `#AEB784` 또는 `#E3DBBB`, 텍스트 `#41431B`, radius 8px, 패딩 충분히
- 상태 뱃지("준비됨"): 작은 dot + 텍스트. 초록색 dot 대신 `#AEB784` dot 사용

---

### 기술 조건

- Tauri 앱의 웹 프론트엔드 (HTML + CSS + JS 또는 React)
- CSS variables로 색상 관리
- 반응형 불필요 (데스크탑 고정 레이아웃)
- 외부 CSS 프레임워크 사용하지 않음 (Tailwind 제외, 직접 CSS 작성)
- 기존 기능 로직은 건드리지 않고 마크업과 스타일만 변경

---

### 결과물 형식

변경된 전체 HTML 구조(또는 React 컴포넌트)와 CSS를 작성해줘.
주석으로 "변경됨", "제거됨", "추가됨" 표시해줘.
